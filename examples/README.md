# Geant4 IcyMoons Examples

This folder contains six standalone examples:

- [example_1](example_1/README.md): amorphous ice, `minimal` logging, one event per energy
- [example_2](example_2/README.md): amorphous ice, `full` logging, one event per energy
- [example_3](example_3/README.md): amorphous ice, multi-threaded, cosine angular distribution, `100` particles at `0.5 MeV`
- [example_4](example_4/README.md): amorphous-ice W-value scan using the current production workflow
- [example_5](example_5/README.md): hexagonal-ice W-value scan using the current production workflow
- [example_6](example_6/README.md): water W-value scan using the current production workflow

Each example has its own:

- macro
- run script
- plotting script or W-value plotting helper
- dedicated `README.md`

Use the README inside each example directory for the exact run and plotting instructions.
